Willkommen zum CSS-Framework Bootstrap!

Bootstrap ist ein freies CSS-Framework und wurde ursprünglich von einem Designer und einem Entwickler bei Twitter ins Leben gerufen. Inzwischen ist es zu einem der berühmtesten Front-End-Frameworks und Open-Source-Projekte auf der ganzen Welt geworden. Das Framework enthält HTML und CSS basierende Gestaltungsvorlagen für Typografie Formulare, Buttons, Tabellen, Grid-Systeme, Navigations und viele weitere Oberflächengestaltungselemente. Zusätzliche optionale JavaScript-Erweiterungen enthält Bootsrap ebenfalls.
